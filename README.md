# DAMMA - Traditional Draughts Game

A fully playable, modern web-based implementation of the traditional Damma (Turkish/Moroccan Draughts) game with local multiplayer support.

**Developed and Maintained by alaeTR**

## Features

- **Multiple Game Variants**: Turkish, Spanish, and Moroccan (Free Mode)
- **Flexible Board Setups**: Standard and Diagonal layouts
- **Smart AI**: Challenge yourself against a Hard AI Engine with adjustable difficulty
- **Local Multiplayer**: Play against a friend on the same device
- **Move Validation**: Enforced rules for valid moves and mandatory captures

## Getting Started

**Prerequisites:** Node.js

1.  **Install dependencies:**
    ```bash
    npm install
    ```

2.  **Start the game:**
    ```bash
    npm run start
    ```
    *(Note: The custom start command `npm run start` is used instead of the default `dev`)*

3.  **Open in Browser:**
    Navigate to [http://localhost:3000](http://localhost:3000) to play.

## Build

To create a production build:

```bash
npm run build
```
